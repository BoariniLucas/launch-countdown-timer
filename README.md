# Launch Countdown Timer

A proposta desse projeto é desenvolver um timer para contagem regressiva para lançamentos de foguetes, além de sua proposta ele po ser utilizado para outras finalizadades também, como por exemplo a contagem regressiva para o dia do eu anivrsário.
